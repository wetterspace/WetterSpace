WetterSpace API

Vorstellung

Wir haben es uns zur Aufgabe gemacht, die Wetterdaten Deutschlands der letzten Jahre für jeden zugänglich zu machen. Das Fundament für dieses Unterfangen bildet unsere WetterSpace API. Dieser Service bezieht regelmäßig die aktuellen Daten des Deutschen Wetterdienstes und bereitet sie so auf, dass sie von Programmierern genutzt werden können. Sie ermöglicht es, auf unkompliziertem Weg, spezifische Wetterdaten für bestimmte Zeiträume zu erhalten.

Aufbau einer Anfrage

Eine Abfrage erfolgt über eine GET- oder Post-Request an folgende URL:
https://www.wetter.space/cgi-bin/jsonGenerator.cgi

Die Parameter für die Abfrage werden dabei an die URL angehängt.Wichtig ist hierbei die richtige Codierung der Abfrage, da bei einigen Elementen ein Leerzeichen vorkommt. In diesem Fall ersetzen Sie das Leerzeichen mit %20. Bei der Abfrage kann entweder eine Postleitzahl oder geografische Koordinaten verwendet werden. Das Ergebnis wird in Form eines JSON Arrays zurückgegeben.

Parameter

Parameter Werte/Richtlinien
location Deutsche Postleitzahl
lat Geografische Breite
long Geografische Länge
start_date Datum im Format: YYYY-MM-DD
end_date Datum im Format: YYYY-MM-DD
element Eines der folgenden Elemente:
  • Lufttemperatur Tagesmaximum
  • Lufttemperatur Tagesminimum
  • Lufttemperatur Tagesmittel
  • Neuschneehöhe
  • Niederschlagshöhe
  • Relative Luftfeuchte
  • Schneehöhe

Errors

Im Fall, dass es auf dem Server zu einem Problem bei der Abfrage kommt, werden die folgenden Error-Nachrichten zurückgegeben. Dabei werden die Elemente zusätzlich angegeben, die die Ursache für den Fehler sind.

  • no data
  • no database
  • no valid zip code
no data Es existieren keine Daten für die angegebenen Parameter
no database Es existiert keine Datenbank für die angegebenen Parameter (location oder lat/long)
no valid zip code Die angegebene Postleitzahl ist nicht gültig

Beispiel

https://www.wetter.space/cgi-bin/jsonGenerator.cgi?location=12277&start_date=2014-07-01&end_date=2014-07-05&element=Lufttemperatur%20Tagesmittel

Diese Anfrage erhält folgende Antwort in Form eines JSON Arrays:

[{"location":"433","wert":"16,2","einheit":"Grad C","date":"2014-07-01","element":"Lufttemperatur Tagesmittel"},
{"location":"433","wert":"16,5","einheit":"Grad C","date":"2014-07-02","element":"Lufttemperatur Tagesmittel"},
{"location":"433","wert":"19,0","einheit":"Grad C","date":"2014-07-03","element":"Lufttemperatur Tagesmittel"},
{"location":"433","wert":"24,0","einheit":"Grad C","date":"2014-07-04","element":"Lufttemperatur Tagesmittel"},
{"location":"433","wert":"23,8","einheit":"Grad C","date":"2014-07-05","element":"Lufttemperatur Tagesmittel"}]
